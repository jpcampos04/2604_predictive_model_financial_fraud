# 💳 Financial Fraud Prediction Model

## 📌 Project Overview
This project focuses on building a machine learning model to detect fraudulent financial transactions. It combines statistical analysis and predictive modeling to address one of the main challenges in fraud detection: class imbalance.

The project also explores practical constraints such as memory limitations (RAM) when working with large-scale datasets.

---

🎯 Objectives
Develop a classification model to predict fraudulent transactions
Apply hypothesis testing and correlation analysis
Handle imbalanced data effectively
Optimize performance under computational constraints
